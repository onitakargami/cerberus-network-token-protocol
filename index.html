Protocol Overview

The Cerberus Network Token Protocol (CRDT), also known as Credit or Cred, is a Solana SPL Token Program built for utility.

The protocol features a fixed token supply that is trustless, immutable and inflation-resistant.

All mint, freeze, and program update authorities have been permanently revoked, ensuring decentralization, automation and trustlessness. A real Solana token program with real blockchain utility.

Tokenomics Fundamentals

Built on Solana's SPL Token Program with a finite supply, the protocol is designed with scarcity, immutability and inflation-resistant properties.

No central authority or backdoor mechanisms - completely immutable, trustless, and verifiable on-chain.

The system is circular by design - supply is released into circulation - no additional tokens can be minted - no single point of authority - no single point of control - no centralization.
